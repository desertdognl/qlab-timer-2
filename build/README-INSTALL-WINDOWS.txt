QLab Timer Windows install notes

If Windows Defender SmartScreen shows a warning for this installer, it is
usually because the app is unsigned.

Install steps:
1. Run "QLab Timer Setup <version>.exe".
2. If SmartScreen appears, click "More info".
3. Click "Run anyway".
4. Complete the installer wizard.

If your antivirus blocks the installer:
- Open your antivirus quarantine/history.
- Restore the installer if you trust the source.
- Add an exception for this installer if needed.

Note:
A fully trusted, warning-free distribution requires code signing.
